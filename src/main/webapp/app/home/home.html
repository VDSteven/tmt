The different features of the time management tool can only be accessed by logged in users.

Once logged in a tab labelled "Time Management Tool" will be available in the navigation bar. Here is a description of the the different features:

  • Projects: a user can view/add/edit projects
  • Manage Time: a user can add his data or the data of users he/she is subsidiary of
  • Statistics: a user can view monthly statistics of holidays, hours and expenses for different users and projects
  • Approve entries: a user can approve the hours and expenses of users he/she is head of
  • Set subsidiary: a user can define one or more subsidiaries for himself/herself
  • Set head(only admin account): an admin can define which user is head of which other users
